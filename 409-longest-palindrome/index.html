409. Longest Palindrome

leetcode
创建时间:2017-11-01 23:59
阅读:
评论:
  1. Longest Palindrome

解题思路：

  1. 找最大回文字符串
  2. 使用 set 的集合不重复的性质
  3. 遍历字符串，不在set中，则放进去
  4. 在set中，计数加2
  5. 结果要考虑字符串长度在 0，1，2 的特殊情况
